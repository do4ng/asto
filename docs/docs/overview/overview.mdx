Asto is a Typescript/Javascript package bundler.  
This is a bundler for the nodejs environment, not a bundler for the browser environment.

esbuild is used by default, but other bundlers can be used through custom loaders.

## Usage

- With javascript

```ts
const { asto } = require('asto');

asto({
  entryPoints: [
    {
      // input file
      input: './src/index.ts',
      // output file
      output: './dist/index.js',
    },
  ],
});
```

- with cli

`asto.config.js` is used as the default configuration file.

```js [title="asto.config.js"]
/**
 * @type {import("asto").BuildOptions[]}
 */
module.exports = {
  // ...
};
```
