`@asto/esm` converts commonjs code to esm code.

> It would be helpful to also look at the [c2es](https://github.com/do4ng/c2es) package.

## Installation

```bash
$ npm i --save-dev @asto/esm
```

## Usage

```js [title="asto.config.js"]
const { esmLoader } = require('@asto/esm');

module.exports = {
  loader: esmLoader(),
  entryPoints: [
    {
      input: 'src/index.js',
    },
  ],
};
```
