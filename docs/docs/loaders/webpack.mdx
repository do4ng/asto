If you want to add more stability you can use the webpack loader.

## Installation

To use the webpack loader, you need to install `webpack` and `@asto/webpack`.

```bash
$ npm i --save-dev @asto/webpack webpack
```

## Usage

```js [title="asto.config.js"]
const { webpackLoader } = require('@asto/webpack');

module.exports = {
  loader: webpackLoader(
    {
      typescript: true,
      nodeExternals: true,
      cli: false,
    },
    {
      /* webpack options */
    },
  ),
};
```

- `typescript`: use `ts-loader`
- `nodeExternals`: use `webpack-node-externals`
- `cli`: add cli banner
