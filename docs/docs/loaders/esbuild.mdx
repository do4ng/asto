The esbuild loader is built into the `asto` package.

```ts
const { esbuildLoader } = require('asto');

module.exports = {
  loader: esbuildLoader(),
  entryPoints: [],
};
```

<details>
<summary><code>`esbuildLoaderOptions`</code></summary>
<br>
This is how you dropdown.
</details>

---

> asto's default loader is esbuild loader.
> So, if you do not set the loader, it will be built with esbuild loader.
